개인정보처리방침

오늘은 남에게 보여주기 위한 기록이 아니라, 오직 나만을 위한 하루를 남기는 개인 기록 앱입니다.

시행일: 2026년 6월 2일

1. 기본 원칙

오늘은 사용자의 사진, 노트, 날짜, 카테고리, 위치정보를 개발자 서버에 저장하지 않습니다. 사용자가 작성한 기록은 사용자의 기기 또는 사용자의 iCloud Drive에 저장됩니다.

개발자는 사용자의 기록 내용, 사진, 위치정보, 노트 내용을 열람하거나 수집하지 않습니다. 또한 오늘은 광고, 추적, 인앱결제, 구독 기능을 제공하지 않습니다.

2. 저장될 수 있는 정보

오늘은 개인 기록 기능을 제공하기 위해 사용자가 직접 입력하거나 선택한 정보를 앱 내부에 저장할 수 있습니다.

  • 사진 또는 이미지 파일
  • 노트, 글귀 등 사용자가 작성한 텍스트
  • 기록 날짜
  • 카테고리 정보
  • 사용자가 선택하거나 허용한 위치정보
  • 사진에 포함된 EXIF GPS 정보가 있는 경우, 기록 위치 제안을 위한 위치정보

위 정보는 사용자가 기록을 작성하고, 캘린더·지도·검색·인사이트 기능으로 다시 확인할 수 있도록 사용됩니다.

3. 정보의 저장 위치

기기 내 저장

iCloud Drive를 사용할 수 없는 경우, 사용자의 기록은 사용자의 기기 내부 저장소에 저장됩니다.

iCloud Drive 저장 및 동기화

iCloud Drive가 활성화된 경우, 기록 JSON 파일과 이미지 파일은 사용자의 iCloud Drive에 저장될 수 있습니다. 이를 통해 동일한 Apple ID를 사용하는 여러 기기에서 기록을 동기화할 수 있습니다.

iCloud Drive에 저장되는 데이터는 사용자의 Apple 계정 및 iCloud 환경을 통해 관리됩니다. 개발자는 사용자의 iCloud Drive에 저장된 기록을 열람하거나 수집하지 않습니다.

개발자 서버 미사용

오늘은 별도의 개발자 운영 서버를 사용하여 사용자의 기록을 저장하지 않습니다.

4. 권한 사용 목적

사진 접근

사용자가 기록에 첨부할 사진을 선택하기 위해 사진 보관함 접근 권한을 사용할 수 있습니다.

위치 접근

사용자가 원할 경우 현재 위치를 기록에 저장하고, 위치가 포함된 기록을 지도에서 다시 보기 위해 위치 접근 권한을 사용할 수 있습니다.

위치정보는 선택 사항입니다. 위치 접근을 허용하지 않아도 사진과 노트 기록 기능은 사용할 수 있습니다.

사진 EXIF 위치정보

사용자가 선택한 사진에 위치정보가 포함되어 있는 경우, 앱은 해당 위치를 기록 위치로 제안하거나 저장할 수 있습니다. 사용자는 지도 핀을 직접 조정하거나 위치 없이 기록을 저장할 수 있습니다.

iCloud Drive

사용자의 여러 기기 간 기록을 동기화하기 위해 iCloud Drive를 사용할 수 있습니다. iCloud Drive를 사용할 수 없는 경우에는 로컬 저장소를 사용합니다.

5. 정보의 이용 목적

오늘은 저장된 정보를 다음 목적을 위해 사용합니다.

  • 하루 기록 작성, 수정, 삭제
  • 기록 상세 화면 표시
  • 월간·연간 캘린더 표시
  • 노트, 카테고리, 날짜 기반 검색
  • 연속 기록, 카테고리 분포 등 인사이트 제공
  • 위치가 포함된 기록의 지도 표시
  • 사용자가 선택한 기록의 공유 카드 이미지 생성
  • 사용자의 기기 간 iCloud Drive 동기화

6. 공유 기능

오늘은 사용자가 직접 선택한 기록을 카드 이미지 형태로 만들어 시스템 공유 시트를 통해 공유할 수 있는 기능을 제공합니다.

공유 기능은 사용자가 직접 실행한 경우에만 동작합니다. 앱은 사용자의 기록을 자동으로 외부 서비스에 업로드하거나 전송하지 않습니다.

공유 시 카드에는 사용자가 작성한 사진, 날짜, 카테고리, 노트, 위치 지명 등이 포함될 수 있으므로, 공유 전 미리보기 화면에서 내용을 확인한 후 공유하시기 바랍니다.

7. 제3자 제공 및 추적

오늘은 사용자의 개인정보를 제3자에게 판매하거나 제공하지 않습니다.

  • 광고 SDK를 사용하지 않습니다.
  • 분석 SDK를 사용하지 않습니다.
  • 사용자 추적을 위한 기능을 사용하지 않습니다.
  • 회원가입이나 로그인 기능을 요구하지 않습니다.

단, 사용자가 시스템 공유 시트를 통해 AirDrop, 메시지, 인스타그램 등 외부 앱으로 공유하는 경우에는 해당 외부 서비스의 개인정보 처리방침이 적용될 수 있습니다.

8. 데이터 삭제 방법

사용자는 앱 내에서 개별 기록을 삭제할 수 있습니다. 기록을 삭제하면 해당 기록의 텍스트, 날짜, 카테고리, 위치정보 및 연결된 이미지가 앱의 저장소에서 삭제될 수 있습니다.

앱을 삭제하면 기기 내부에 저장된 로컬 데이터가 삭제될 수 있습니다. iCloud Drive에 저장된 데이터는 사용자의 iCloud Drive 설정 또는 파일 관리 방식에 따라 별도로 남아 있을 수 있으며, 사용자가 직접 iCloud Drive에서 삭제할 수 있습니다.

9. 아동의 개인정보

오늘은 별도의 회원가입, 계정 생성, 온라인 커뮤니티 기능을 제공하지 않습니다. 또한 개발자는 사용자의 기록 내용을 수집하거나 열람하지 않습니다.

보호자가 필요하다고 판단하는 경우, 기기 및 iCloud 설정을 통해 앱 사용과 데이터 저장을 관리할 수 있습니다.

10. 보안

오늘은 사용자의 기록을 개발자 서버에 저장하지 않고, 사용자의 기기 또는 사용자의 iCloud Drive에 저장합니다. 사용자는 기기 잠금, Apple ID 보안, iCloud 보안 설정을 통해 기록을 보호할 수 있습니다.

다만, 사용자의 기기 분실, Apple ID 접근 권한 노출, 외부 앱으로의 공유 등 사용자의 관리 환경에 따라 데이터가 노출될 수 있으므로 주의가 필요합니다.
