개인정보처리방침

오늘 (Oneul) · 김규민

오늘 앱은 사용자의 소중한 하루를 기록하기 위한 개인 일기 앱입니다.
수집되는 모든 데이터는 오직 사용자의 기기 및 개인 iCloud 계정에만 저장되며, 외부 서버로 전송되거나 제3자와 공유되지 않습니다.

1. 수집하는 정보

앱은 다음 정보를 수집합니다.

  • 사진 — 사용자가 직접 선택한 사진 1장 (일기 기록용)
  • 위치 정보 — 사진의 GPS 메타데이터 또는 현재 위치 (선택 사항, 사용자 동의 시)
  • 텍스트 — 사용자가 직접 입력한 하루의 감상 및 카테고리
  • 날짜/시간 — 기록 날짜

2. 정보 이용 목적

수집된 정보는 오직 다음 목적으로만 사용됩니다.

  • 일기 기록의 저장 및 표시
  • 캘린더 및 지도 기능 제공
  • iCloud를 통한 기기 간 데이터 동기화 (사용자 개인 iCloud 계정 한정)

3. 데이터 저장 위치

모든 데이터는 다음 위치에만 저장됩니다.

  • 기기 내부 저장소 — 앱의 로컬 Documents 폴더
  • 사용자 개인 iCloud Drive — iCloud 동기화 활성화 시 (Apple의 보안 인프라 사용)

개발자 또는 제3자 서버에는 어떠한 데이터도 저장되지 않습니다.

4. 제3자 공유

수집된 개인정보는 어떠한 제3자에게도 판매, 임대, 공유되지 않습니다.

5. 접근 권한

앱은 다음 권한을 요청할 수 있습니다.

  • 사진 라이브러리 — 기록에 첨부할 사진 선택 시
  • 위치 정보 — 기록에 위치를 추가할 때 (선택 사항)

각 권한은 해당 기능 사용 시에만 요청되며, 허용하지 않아도 기본 기능은 정상 사용 가능합니다.

6. 데이터 삭제

앱을 삭제하면 기기에 저장된 모든 데이터가 삭제됩니다. iCloud에 저장된 데이터는 iPhone 설정 → Apple ID → iCloud → 앱 목록에서 별도로 삭제할 수 있습니다.

7. 아동 개인정보 보호

이 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

8. 방침 변경

개인정보처리방침이 변경될 경우 앱 업데이트를 통해 공지합니다. 변경된 방침은 공지일로부터 효력이 발생합니다.
