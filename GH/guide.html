User Guide

재무의사결정지원 시스템
사용 안내

GH 경기주택도시공사 중장기 재무전망 분석 플랫폼의 주요 기능을 안내합니다.

시스템 접속
시작하기
로그인 후 아래 순서로 진행하세요.
1

로그인

발급된 이메일과 비밀번호로 접속합니다.

2

프로젝트 선택

왼쪽 사이드바의 활성 프로젝트 드롭다운에서 분석할 프로젝트를 선택합니다.

3

메뉴 선택

상단 내비게이션에서 원하는 분석 메뉴를 클릭합니다.

4

차트 / 데이터 확인

각 페이지 상단의 차트 · 데이터 탭으로 전환하며 분석합니다.

메뉴별 기능
사이드바의 재무전망 메뉴 5개를 소개합니다.
중장기재무전망
매출·이익·부채·자본·선수금·금융부채·부채비율 등 핵심 재무 지표를 연도별로 시각화합니다.
실적 연도(★)와 전망 연도를 색상으로 구분합니다.
매출 / 영업이익 / 순이익 부채 / 자본 / 금융부채 부채비율 추이 단위 변경 엑셀 다운로드
자금수지전망
연도별 수입·지출과 순자금(수입−지출)을 분석합니다. 수입·지출 세부 구성 및 사업수지를 세분화 차트로 확인할 수 있습니다.
수입 / 지출 / 순자금 수입·지출 세부 구성 사업수지 단위 변경 엑셀 다운로드
재무비율전망
부채비율·차입금의존도·이자보상배율 등 주요 재무 비율을 꺾은선 차트로 전망합니다. 위험 기준선(예: 부채비율 200%) 참조선이 표시됩니다.
부채비율 / 차입금의존도 이자보상배율 / 영업이익률 유동비율 등 다수 지표 단위 변경 엑셀 다운로드
부채전망
금융부채(금융차입·정부기금 등)와 비금융부채(선수금·임대보증금 등) 세부 구성을 분석합니다. 총부채 추이와 최고치 연도를 KPI 카드로 한눈에 파악할 수 있습니다.
금융부채 세부 구성 비금융부채 세부 구성 총부채 추이 최고치 KPI 카드 단위 변경 엑셀 다운로드
프로젝트별 비교
두 프로젝트(예: 기본안 vs 시나리오1)를 나란히 비교합니다. 매출·순이익·부채비율·자금수지 4개 KPI 차이를 요약하고, 항목별 비교 차트를 제공합니다.
A vs B 프로젝트 비교 4개 KPI 요약 카드 경영성과 / 재무상태 비교 엑셀 다운로드
공통 기능
모든 분석 메뉴에서 사용할 수 있는 기능입니다.

차트 / 데이터 탭

각 메뉴 상단의 탭으로 시각화 차트와 원시 데이터 표를 전환합니다.

범례 클릭 토글

차트 범례 항목을 클릭하면 해당 계열을 숨기거나 다시 표시할 수 있습니다.

단위 변경

데이터 탭에서 억원 / 백만원 / 천원 단위를 즉시 전환합니다. 엑셀 다운로드 시 선택한 단위로 저장됩니다.

실적 / 전망 구분

프로젝트 기준연도 이전은 실적(★), 이후는 전망으로 자동 구분합니다. 차트에서 회색 바로 표시됩니다.

엑셀 다운로드

각 메뉴의 데이터를 서식이 적용된 .xlsx 파일로 다운로드합니다.

프로젝트 전환

사이드바 드롭다운에서 프로젝트를 변경하면 모든 메뉴가 즉시 해당 프로젝트 데이터로 전환됩니다.

데이터 입력 — 엑셀 업로드
관리자가 분기별 전망 데이터를 업로드하는 방법입니다.
  1. 사이드바 하단 엑셀 업로드 메뉴로 이동합니다.
  2. 사이드바에서 업로드할 프로젝트를 먼저 선택합니다.
  3. 정해진 형식의 엑셀 파일을 드래그하거나 클릭하여 선택합니다.
  4. 각 시트의 미리보기를 확인한 후 업로드 실행 버튼을 누릅니다.
  5. 업로드 완료 메시지 확인 후 분석 메뉴에서 데이터를 확인합니다.
시트명 (포함 문자) 업로드 대상 메뉴 주요 컬럼 구조
재무전망 중장기재무전망 항목명 | 연도별 금액
자금수지 자금수지전망 구분1 | 구분2 | 연도별 금액
재무비율 재무비율전망 항목명 | 연도별 비율
부채전망 부채전망 구분1 | 구분2 | 연도별 금액
참고사항

· 동일 프로젝트에 데이터를 재업로드하면 기존 데이터가 전체 대체됩니다.
· 엑셀 파일의 시트명에 위 키워드가 포함되어 있어야 정상 인식됩니다.
· 데이터가 표시되지 않을 경우 사이드바에서 프로젝트 선택 여부를 먼저 확인하세요.
· 문의사항은 시스템 관리자에게 연락하세요.
