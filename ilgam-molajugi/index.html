일감몰아주기 과세
적용대상 판단표와 계산예시

이 페이지는 “어떤 법인이 일감몰아주기 계산자료를 만들어야 하는지”를 먼저 판단하고, 그 다음 주주별 신고대상 여부와 증여의제이익 계산 구조를 예시로 이해할 수 있도록 구성했습니다.

적용대상 판단법인별 판단표주주별 판단계산예시올해 신고과세제외자료

1. 어떤 법인이 일감몰아주기 검토를 해야 하나?

핵심: 일감몰아주기 증여세는 형식상 수혜법인이 내는 세금이 아니라 지배주주·친족 개인이 신고하는 증여세입니다. 하지만 판단에 필요한 매출·이익·주주자료는 수혜법인이 가지고 있으므로, 아래 요건에 걸리는 법인은 계산자료를 준비해야 합니다.
순서질문YES이면NO이면
1수혜법인이 내국법인인가?다음 단계로 진행일감몰아주기 수혜법인 검토대상에서 벗어날 가능성 큼
2해당 사업연도에 세후영업이익이 있는가?실제 증여의제이익 발생 가능계산상 증여의제이익은 보통 0. 다만 검토표는 보관 권장
3수혜법인에 지배주주가 있는가?지배주주 기준 특수관계법인을 판정지배주주 특정이 어렵다면 최대주주등 구조를 먼저 정리
4지배주주와 특수관계에 있는 법인에 대한 매출이 있는가?거래처별 특수관계법인 매출액 집계적용대상에서 벗어날 가능성 큼

6지배주주와 친족 중 주식보유 기준을 초과하는 개인이 있는가?그 개인별로 증여의제이익 및 증여세 신고 검토거래비율 요건을 충족해도 개인 신고대상은 없을 수 있음

2. 법인별 적용대상 판단표

아래 표는 실무자가 회사별로 “계산자료 작성 필요 여부”를 빠르게 판단하기 위한 표입니다. 각 회사별로 한 줄씩 체크하면 됩니다.

법인 유형먼저 볼 것거래비율 요건주주 보유율 요건판단 결과
일반기업특수관계법인 매출 규모와 비율을 모두 확인원칙: 30% 초과
추가: 특수관계법인 매출 1,000억원 초과 & 거래비율 20% 초과		지배주주와 친족 개인의 직·간접 보유율 3% 초과일반기업은 1,000억원 특례 때문에 30% 이하라도 별도 검토 필요
중견기업중견기업 해당 여부와 과세제외매출40% 초과지배주주와 친족 개인의 직·간접 보유율 10% 초과거래비율 40% 초과 + 보유율 10% 초과 개인이 있으면 계산 필요
중소기업상대방도 중소기업인지, 과세제외 가능 여부50% 초과지배주주와 친족 개인의 직·간접 보유율 10% 초과중소기업 간 거래 등 제외 후에도 50% 초과인지 확인
세후영업손실 법인세후영업이익이 0 이하인지거래비율을 초과해도 계산상 이익은 0 가능주주요건은 참고 검토대체로 납부세액 없음. 단, 근거자료 보관 권장
특수관계 매출 없는 법인주요 매출처와 지배주주 특수관계 여부해당 없음해당 없음적용대상 아님 가능성 큼
특수관계 매출은 있으나 주주 보유율 미달거래비율과 주주별 직접·간접 보유율충족할 수 있음일반 3%, 중견·중소 10% 이하수혜법인 요건은 있어도 해당 개인 과세는 없을 수 있음
수혜법인 계산 필요성 = 내국법인 + 세후영업이익 양수 + 지배주주 존재 + 특수관계법인 매출 존재 + 거래비율 기준 초과

3. 주주별 신고대상 판단

법인이 적용대상일 수 있어도, 실제 증여세 신고는 지배주주와 그 친족 개인별로 판단합니다.

지배주주수혜법인의 최대주주등 중 직접·간접 보유비율이 가장 높은 개인. 법인이 최대주주이면 그 법인을 통해 최종 개인까지 올라가 봅니다.
수증자 후보지배주주와 친족 관계에 있는 개인주주
보유율 기준일반기업: 3% 초과 / 중견·중소기업: 10% 초과
보유율 계산직접보유율 + 간접보유율. 중간법인이 있으면 단계별 지분율을 곱해서 간접보유율을 계산합니다.

주주별 판단표 양식

주주관계직접%간접%판정
지배주주 본인본인20%0%기준 초과 여부 확인
배우자친족2%0%일반기업이면 3% 이하로 제외 가능
자녀친족8%5%합산 13%로 기준 초과 가능
법인주주 A법인40%-개인 수증자 아님. 다만 개인의 간접보유 계산에 사용

4. 계산 예시

예시 1: 일반기업, 과세 가능

사실관계

  • 수혜법인 총매출액: 1,000억원
  • 특수관계법인 매출액: 350억원
  • 과세제외매출: 없음
  • 세후영업이익: 50억원
  • 자녀 보유율: 직접 5%

판단

  1. 거래비율 = 350억원 / 1,000억원 = 35%
  2. 일반기업 기준 30% 초과 → 수혜법인 요건 충족
  3. 자녀 보유율 5% → 일반기업 주식보유 기준 3% 초과
  4. 조정거래비율 = 35% - 5% = 30%
  5. 조정주식보유비율 = 5%
  6. 증여의제이익 = 50억원 × 30% × 5% = 7,500만원
증여의제이익 75,000,000원

예시 2: 중소기업, 과세제외 후 제외

사실관계

  • 총매출액: 100억원
  • 특수관계법인 매출액: 60억원
  • 그중 중소기업 간 거래로 과세제외 가능: 30억원
  • 세후영업이익: 5억원
  • 자녀 보유율: 20%

판단

  1. 과세제외 후 특수관계 매출 = 60억원 - 30억원 = 30억원
  2. 과세제외 후 총매출 = 100억원 - 30억원 = 70억원
  3. 거래비율 = 30억원 / 70억원 = 42.86%
  4. 중소기업 기준 50% 이하 → 수혜법인 요건 미충족
거래비율 요건 미충족 → 주주 보유율이 높아도 과세대상 아님

예시 3: 일반기업 1,000억원 특례

사실관계

  • 총매출액: 6,000억원
  • 특수관계법인 매출액: 1,300억원
  • 거래비율: 21.67%
  • 세후영업이익: 200억원
  • 배우자 보유율: 4%

판단

  1. 거래비율은 30% 이하라서 원칙 기준은 미달
  2. 하지만 특수관계법인 매출액 1,000억원 초과
  3. 거래비율 20% 초과
  4. 일반기업 특례 요건 검토 필요
  5. 배우자 보유율 4%로 일반기업 3% 초과
30% 이하라도 적용대상 가능성 있음

예시 4: 거래비율은 높지만 주주요건 미달

사실관계

  • 중견기업 총매출액: 500억원
  • 특수관계법인 매출액: 250억원
  • 거래비율: 50%
  • 세후영업이익: 20억원
  • 친족 개인 보유율: 8%

판단

  1. 중견기업 거래비율 기준 40% 초과 → 수혜법인 요건 가능
  2. 하지만 중견기업 주식보유 기준은 10% 초과
  3. 친족 보유율 8% → 수증자 요건 미충족
수혜법인 검토는 필요하지만 해당 개인 증여세는 없을 수 있음

5. 2026년 신고분 체크

대상 사업연도통상 2025사업연도분을 2026년에 신고합니다.
신고·납부기한수혜법인의 법인세 과세표준 신고기한이 속하는 달의 말일부터 3개월. 12월말 법인은 2026.6.30.
신고세액공제기한 내 신고 시 산출세액의 3%.
지방세증여세에는 지방소득세 10%가 별도로 붙지 않습니다.

증여세 계산 구조

증여의제이익 = 세후영업이익 × 조정 특수관계법인거래비율 × 조정 주식보유비율 - 배당소득 공제액
일반기업거래비율 - 5%, 주식보유비율 차감 없음
중견기업거래비율 - 20%, 주식보유비율 - 5%
중소기업거래비율 - 50%, 주식보유비율 - 10%

6. 과세제외매출 적용 포인트

구분검토 포인트증빙
중소기업 간 거래수혜법인과 거래상대방이 모두 중소기업인지 확인중소기업확인서, 거래처별 매출명세
자회사·지분관계 거래지분율, 지배관계, 보유기간 확인주주명부, 지분구조도
수출 목적 거래실제 수출과 연결되는 거래인지 확인수출신고필증, 인보이스, 선적서류
법률상 의무거래법령·인허가 조건에 따른 의무거래인지 확인근거 법령, 공문, 계약서

필수 검토자료

  • 수혜법인 2025사업연도 재무제표와 세후영업이익 계산자료
  • 거래처별 매출명세와 지배주주 기준 특수관계법인 판단표
  • 과세제외매출 증빙: 중소기업확인서, 수출서류, 법령상 의무 근거, 지분관계 자료
  • 2025사업연도 종료일 현재 주주명부
  • 지배주주 및 친족관계 확인자료
  • 중간법인 주주구조와 간접보유율 계산표
  • 신고기한 내 배당소득 발생 및 귀속 자료

신고 전 오류 방지

  • 법인세 신고기한 기준으로 증여세 신고기한 산정
  • 수혜법인 요건과 수증자 요건을 분리해서 판단
  • 특수관계법인 여부를 지배주주 기준으로 판단
  • 과세제외매출을 분자·분모에서 일관되게 차감
  • 일반기업 1,000억원·20% 규칙 별도 확인
  • 지방소득세 별도 납부를 잘못 추가하지 않기
