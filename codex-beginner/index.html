완전 초보용 Step-by-Step

코덱스로 시작하는 코딩 학습 + 깃허브 페이지 배포

이 페이지는 “아무것도 모르는 상태”를 기준으로 만들었습니다. 그대로 따라 하면 작은 프로젝트를 만들고, GitHub에 올리고, GitHub Pages로 공개하는 것까지 한 번에 경험할 수 있어요.

학습 원칙: 작게 만들기 → 바로 실행해 보기 → 반복해서 각인하기.

목차

  1. 시작 전 준비
  2. 코덱스에게 질문하는 기본 공식
  3. 프로젝트 1: 자기소개 HTML 페이지
  4. 프로젝트 2: 할 일 목록(To-do) 페이지
  5. 프로젝트 3: 간단 계산기 페이지
  6. Git/GitHub 업로드 순서
  7. GitHub Pages 공개(배포) 순서
  8. 각인용 반복 루틴(7일)

1) 시작 전 준비

Step 1. GitHub 계정 만들기 (이미 있으면 생략)
Step 2. 코드 에디터 설치 (VS Code 추천)
Step 3. Git 설치 후 터미널에서 아래 확인 
git --version
Step 4. 새 폴더 만들기 (예: my-first-web)

2) 코덱스에게 질문하는 기본 공식

초보일수록 “정확한 요청 템플릿”이 중요합니다.

[역할] 너는 코딩 선생님
[목표] 초보가 따라할 수 있는 HTML/CSS/JS 예제 만들기
[조건] 파일 구조, 전체 코드, 실행 방법, 오류 대처 포함
[출력] 복붙 가능한 코드 + 단계별 설명

잘 안되면 “오류 메시지 전체”를 그대로 붙여 넣고 물어보세요.

3) 프로젝트 1: 자기소개 HTML 페이지

목표: HTML 태그에 익숙해지기.

1. index.html 파일 생성
2. 제목, 문단, 링크, 이미지 태그 넣기
3. 브라우저에서 파일 열어 보기
<h1>안녕하세요! 저는 코딩 초보입니다.</h1>
<p>하루 30분씩 코딩을 연습합니다.</p>
<a href="https://github.com" target="_blank">내 GitHub 가기</a>

4) 프로젝트 2: 할 일 목록(To-do) 페이지

목표: JavaScript로 화면 반응 만들기.

핵심 기능

  • 입력창에 할 일을 쓰고 추가 버튼 클릭
  • 목록에 항목이 표시됨
  • 삭제 버튼으로 제거

코덱스 요청 예시

HTML/CSS/JS 한 파일로 동작하는 초보용 투두 앱을 만들어줘.
버튼 클릭으로 추가/삭제가 되게 하고,
코드를 줄 단위로 주석 달아줘.

5) 프로젝트 3: 간단 계산기 페이지

목표: 함수, 이벤트, 숫자 처리 감각 익히기.

1. 숫자 입력 2개 + 연산 버튼(+, -, ×, ÷) 만들기
2. 클릭 시 결과 표시하는 함수 작성
3. 0으로 나누기 예외 메시지 처리

6) Git/GitHub 업로드 순서

# 프로젝트 폴더에서 실행

git init
git add .
git commit -m "first project"
git branch -M main
git remote add origin https://github.com/내아이디/내저장소.git
git push -u origin main

처음 push에서 로그인/토큰 인증이 필요할 수 있습니다.

7) GitHub Pages 공개(배포) 순서

1. GitHub 저장소 접속 → Settings
2. 왼쪽 메뉴 Pages 클릭
3. Source: Deploy from a branch
4. Branch: main / Folder: /root 선택 후 저장
5. 1~3분 뒤 배포 URL 접속: https://아이디.github.io/저장소명

8) 각인용 반복 루틴 (7일)

  • Day 1~2: HTML 태그 10개 직접 타이핑
  • Day 3~4: To-do 앱을 처음부터 다시 제작
  • Day 5: 계산기에서 기능 1개 추가(예: 초기화 버튼)
  • Day 6: Git 명령어를 안 보고 입력
  • Day 7: GitHub Pages 새 저장소에 다시 배포

핵심은 반복 + 작은 성공입니다. 막히는 부분은 코덱스에게 오류 메시지와 함께 질문하세요.
